안심번호 서비스 안내

닫기
  • 주문에서부터 배송완료 기간까지 고객님의 휴대폰 번호가 안심번호로 판매처/택배사에 전달됩니다.
  • 휴대폰번호가 외부에 노출되지 않기 때문에 개인 정보를 보호받으실 수 있습니다.
  • 발급된 안심번호는 일정기간 이후 해지되며, 해지이후 취소/반품등의 절차 진행중에는 실제 연락처가 제공될 수 있습니다.
  • 본 서비스는 LG U+에서 위탁해 무료로 제공해 드립니다.

취소/교환/환불안내

닫기
  • 티켓 구매 후 확송요청 전까지는 구매취소가 가능합니다.
  • 티켓상품일 경우 일부 상품 환불이 불가하여 환불을 원하실 경우 전체상품을 환불하셔야 합니다.
  • 교환의 경우 제품하자로 인한 일부 상품 교환만 가능 합니다.
  • 취소나 환불요청과 요청에 대한 상세한 내역은 마이티몬 에서 확인하실 수 있습니다.
