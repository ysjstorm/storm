환불/교환 요청 가능 기간

구매자 단순 변심일 경우
환불, 교환 요청은 배송완료 시점으로부터 7일 이내에 가능합니다.
상품이 표기 광고의 내용과 다르거나 상품 하자일 경우
상품 수령 후 3개월 이내, 그 사실을 알 수 있었던 날로부터 30일 이내에 환불, 교환 요청이 가능합니다.
단, 티켓 이용권 상품의 경우 구매 후 발송요청 전까지는 환불 요청시 즉시 환불처리 됩니다.

횐불/교환 요청 방법

1. 요청방법: 마이티몬 내 구매내역에서 환불이나 교환 요청
2. 반품방법: 택배기사 반품 상품 수거
반품 상품은 환불 요청 후 1~3일내(주말/공휴일 제외)에 택배기사가 직접 방문합니다.
편의점 반품은 가까운 편의점(CU, GS25)에 방문하셔서 직접 접수합니다.
3. 반품 상품 확인 후 환불/교환처리
판매자가 반품된 상품 확인 후 환불이나 교환처리 됩니다.
교환 상품이 재고가 없는 경우 고객 확인 후 환불처리 될 수 있습니다.

반품 배송비

※ 고객 변심에 의한 반품도 무료로 처리됩니다.

횐불/교환 기준

환불/교환 불가 사유
환불/교환 요청 가능 기간이 초과한 경우
(배송 완료 시점으로부터 7일까지 요청 가능)
고객 주문 확인 후 상품 제작에 들어가는 경우
구성품 분실 및 취급 부주의로 인한 파손, 고장, 오염 등이 발생한 경우
모니터의 해상도 차이로 인하여 색상 등 이미지가 실제와 상이한 경우
구매자의 상용 및 일부 소비로 인해 상품의 가치가 현저히 감소한 경우
시간의 경과에 의해 재판매가 어려울 정도로 상품가치가 현저히 감소한 경우
※ 교환은 제품하자로 인한 동일 상품으로의 교환만 가능합니다.
티켓상품의 환불, 교환 기준
티켓으로 구매한 상품은 세트 상품으로 일부가 아닌 전체 상품을 환불처리 하셔야 합니다.
교환은 제품하자로 인한 동일 상품으로의 교환만 가능합니다.
재고 부족으로 인해 교환이 불가능한 경우 고객 확인 후 환불처리 될 수 있습니다.
